### Jake's Build Tools adds a plethora of different tools to help the player in building awesome things! From experience storage to capturing mobs, this pack does a whole lot! And new things are always being added!

# Features
- Mining Helmet - A glowing hat that also protects the player from falling blocks and anvils!
- Block Wrench - A tool that can rotate any rotatable block! Even ones with inventories!
- Hammer - A tool that can break blocks in a 3x3 area!
- Void Bundle - A bundle that deletes any item put inside it! The last item destroyed can be retrieved by right clicking with it in the player's hand!
- Item Magnet - Attracts items in a 7 block radius towards the player when in their mainhand or offhand! Can also be used in item frames!
- Experience Flask - A method of storing experience! Can be filled with experience and then smashed to get it back!
- Tape Measure - A tool for measuring distances between blocks!
- Rope Ladder - A new type of ladder that places ladders below it!
- Trowel - Places blocks randomly from the player's hotbar, great for texturing!
- Chisel - Replaces a block with the block in the slot to the right of it!
- Hardhat - A snazzy hat that increases the player's reach by 2 and protects them from falling blocks and anvils!
- Mob Bag - Capable of safely storing one non-hostile mob! (includes villagers)
- Void Bucket - A bucket that destroys any liquid put inside of it!
- Grass Starter - Can turn dirt into grass and vice versa!

For details on function and recipes, check out the [wiki](https://github.com/maybejake/Jakes-Build-Tools/wiki)!

# FAQ
### Where is the resource pack?
It's packaged with the datapack! The zip file you download can be used interchangeably, so just drop it into your resourcepack and datapack folders and you'll be good to go!
### Why can't I see the items in the creative menu or in REI/JEI/NEI?
This is a datapack with re-packaged mod versions for convenience of installation. It is currently not possible for custom items added by datapacks to be added to the creative menu or detected by any REI-like mod.
### What are the recipes?
The datapack adds the recipes to your crafting table recipe book by default, you can find them all there. Otherwise, check out the [wiki](https://github.com/maybejake/Jakes-Build-Tools/wiki)!
